How to Import a Generated Practice Set from Bible Fact-Pak™

You will need an account at www.biblefactpak.com

Once you log into Bible Fact-Pak™ click 'Quiz'. You should be on the 'Question Set Generator' page.

You can use an existing practice set or generate a new practice set.

On 'Printing Options' disable all options. Then click 'Print / Preview Version'

In preview output, select and copy (ctrl-c) the text for the questions. Do not select the headers or footers.

Paste the text into the textbox on Quiz Master's Apprentice. Then click 'Parse Text'
